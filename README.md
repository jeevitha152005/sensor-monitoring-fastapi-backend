# Sensor Monitoring FastAPI Backend

This project is a backend service developed using FastAPI.

## Features
- Receives sensor data using API
- Stores sensor data in database
- Provides API to fetch sensor data
- Generates alerts when threshold values are exceeded

## Technologies Used
- Python
- FastAPI
- MySQL
- MQTT (concept)

## API Endpoints
GET /sensors - Get all sensor data

POST /add_sensor - Add sensor data

## Sample Data
Sensor parameters:
- temperature
- humidity
